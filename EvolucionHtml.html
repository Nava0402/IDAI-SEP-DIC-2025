Evolución de HTML

Antes de comenzar debemos de saber primero, ¿Qué es HTML?
HTML son las siglas de Hiper Text Markup Language, esto es, un lenguaje de marcado que permite estrucutrar y organizar nuestro texto de manera legible.

Este sistema inicio en la década de los 90, con el físico Tim Berners Lee, quien era investigador en la Organización Europea para la Investigación Nuclear, en aquel tiempo realizó una publicación con el título HTML Tags en el cual daba una respuesta a la necesidad de crear un sistema de hipertexto que les permitiera a los científicos e investigadores del CERN intercambiar información de forma rápida y fácil de referenciar, esta propuesta se concretó unos años después.

Primera versión: 1993

Esta versión contenía un conjunto básico de etiquetas, que permitían a los desarrolladores crear documentos estructurados con enlaces y formatos de texto simples, algunas de estas etiquetas incluían p para párrafos, h1 para encabezados, a para enlaces. Esto sentó las bases para la estructura básica de las páginas web modernas. Era algo revolucionario pero, conforme la web avanzaba, también HTML debía avanzar principalmente por la gran demanda de desarrolladores y usuarios.

Segunda versión: 1995

Después del gran logro que se había conseguido, se implementaron mejoras al sistema como por ejemplo: diferentes etiquetas. Algunas de etiquetas como ul y ol para crear listas sin ordenar y ordenadas. También se agregaron las etiquetas li para elementos de lista y h2 a h6 para encabezados de diferentes niveles.
Esta versión 2.0 fue la primera propuesta oficial de estandarización de HTML.

Tercera versión: 1997

Luego del crecimiento y siendo respaldada por la World Wide Web Consortium o también conocida como W3C, se agregan más funciones como: soporte para tablas, frames y hojas de estilos en línea. Introdujo nuevas capacidades como applets de Java, scripts y más opciones de presentación. Esta versión marcó el inicio de una web más interactiva y visualmente atractiva.

Cuarta versión: 1999

Aquí pudimos tener nuevas etiquetas como: strong para resaltar texto y em para enfatizarlo. También se incluyó un mejor soporte para el diseño de tablas y la creación de formularios interactivos , con las etiquetas de form, input, button. Hubo además separación de contenido y presentación, introduciendo hojas de estilo en cascada (CSS) para un control más preciso del diseño.

XHTML: 2000

Una versión intermedia que significa Extensible Hyper Text Markup Language, y que se basó en una versión más estricta de HTML que seguía las reglas del estándar XML. Esto permitía un mejor manejo entre diferentes plataformas y dispositivos, pero también implicaba que las páginas web debían cumplir con una sintaxis más rigurosa.

Quinta versión: 2014

Finalmente, nos encontramos con la última actualización de este lenguaje que, ha revolucionado la forma en que se crea y se experimenta el contenido web. HTML5 ha introducido nuevas etiquetas y funciones, como video y audio para reproducir contenido multimedia, canvas para dibujar gráficos en tiempo real, y section y article para estructurar el contenido de una página.
HTML5 es compatible con otros lenguajes web, como CSS que ya se mencionó y JavaScript, que se utilizan para controlar la presentación y la interactividad de las páginas web.
